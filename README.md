## 🏠 AI Interior Design Agent

## 🚀 Overview
This project is an AI-powered interior design agent that helps generate room design suggestions based on user preferences like style, budget, and room size.

## 🎯 Features
- Simulated AI environment for interior design
- Reward-based grading system
- Supports multiple styles (modern, luxury, minimal)
- React frontend for user interaction

## 🧠 How It Works
The agent takes actions (design choices) and receives rewards based on how well they match user constraints.
